Simpan gambar header yang Anda lampirkan di workspace ini dengan nama "header.jpg" di lokasi yang sama dengan file ini.

Contoh path: images/header.jpg

Jika Anda membuka halaman di browser dan gambar tidak muncul, pastikan file gambar benar-benar bernama header.jpg (case-insensitive di Windows) dan berada di folder images.

Anda dapat mengganti gambar dengan file lain kapan saja selama nama file tetap header.jpg.
